Hub-Plugin
==========

Plugin Hub of PlayCraft Networks

Functions:
- Commands:
- /fake (False leave message)
- /fw (Shoot a firework!)
- /rconfig (Reload the config files)

Join and Leave messages (Editable in config.yml):
- &8[&6Hub&8] %user% &7left the game.
- &8[&6Hub&8] %user% &7Joined the game.
